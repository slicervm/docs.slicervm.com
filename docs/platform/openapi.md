# OpenAPI Specification

The Slicer REST API is documented as an OpenAPI 3.0 spec. Use it to generate clients, explore endpoints, or import into tools like Postman.

* [Download openapi.yaml](/openapi.yaml)

To explore interactively, paste the file into [Swagger Editor](https://editor.swagger.io/) or import it into [Postman](https://www.postman.com/).

For the hand-written API reference with examples, see the [REST API reference](/reference/api/).
